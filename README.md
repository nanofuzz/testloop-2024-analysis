# TestLoop analysis

## Prerequisites

### Python

Install [Python][], then install our dependencies:

```sh
pip install -r requirements.txt
./getfonts.sh
```

## Usage

### Python

Run the script:

```sh
python3 main.py
```

This will output the non-marginal contents of the data tables from the article.



[python]: https://docs.python.org/3/using/index.html
